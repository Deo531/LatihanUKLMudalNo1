Saya menggunakan scanner untuk menginputkan panjang, lebar, tinggi, jarak, berat.

Setalah kita menginputkan tersebut, kita bisa menghitung panjang, lebar, tinggi untuk menemukan volume barang
dengan rumus :
panjang * lebar * tinggi


Saya Menggunakana 2 IF
Pertama untuk jarak :
kenapa saya kali dengan berat karena ada intruksi "Jarak Lebih dari 10km =  Rp 6000 per kg" dan "Jarak kurang atau sama dengan 10 km = Rp 4250 per kg"

Kedua untuk volume :
Volume ini digunakan untuk mengecek apakah volume melebihi 100cm^3 atau tidak, kalau iya maka akan dikenakan biaya Rp 50.000

hasil dari if saya masukkan ke variabel hasil dengan +=, agar bisa langsung menambah. Jangan lupa untuk menambahkan hasil = 0 di tipe data agar bisa menghitung
